# Game User Dashboard

Aplikasi website Public API bertema gaming dengan tampilan dark mode.

## Fitur
- Menggunakan Public API
- Dark Mode
- Search Player
- Responsive Design
- Error Handling
- Tampilan Card Gaming
- Live Fetch API

## API
https://jsonplaceholder.typicode.com/users

## Cara Menjalankan
1. Extract ZIP
2. Buka folder project
3. Klik 2x index.html

## Teknologi
- HTML
- CSS
- JavaScript
- Fetch API
