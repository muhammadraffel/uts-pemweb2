# 🚀 Apenkstore - Modern E-Commerce Website

Apenkstore adalah website e-commerce modern dengan desain futuristik bertema gaming yang dibuat menggunakan HTML, CSS, dan JavaScript Vanilla serta menggunakan API online untuk mengambil data produk secara realtime.

Website ini memiliki tampilan premium dengan dark mode & light mode modern, responsive di semua perangkat, serta fitur-fitur seperti search produk, filter kategori, detail produk, dan keranjang belanja interaktif.
